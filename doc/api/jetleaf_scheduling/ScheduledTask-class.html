ScheduledTask

ScheduledTask class abstract interface

Represents a scheduled task that can be queried and canceled.

This interface provides control over a scheduled task's lifecycle, allowing you to check its status and cancel it if needed.

Example:

final task = await scheduler.schedule(
  () => print('Task'),
  trigger,
);

// Check if canceled
if (!task.getIsCanceled()) {
  // Cancel the task
  await task.cancel();
}
