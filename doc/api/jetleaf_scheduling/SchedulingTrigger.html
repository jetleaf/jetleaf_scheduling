SchedulingTrigger

SchedulingTrigger typedef

SchedulingTrigger = ({String name, Trigger trigger})

A record type representing a JetLeaf scheduling trigger configuration.

It pairs a Trigger object, defining the scheduling conditions (like cron or fixed-rate), with an optional name that serves as a descriptive identifier.

Example:

final trigger = (trigger: CronTrigger('0 0 * * *'), name: 'dailyMidnightJob');
print(trigger.name); // dailyMidnightJob
