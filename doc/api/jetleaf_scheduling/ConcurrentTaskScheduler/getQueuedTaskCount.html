getQueuedTaskCount method

int getQueuedTaskCount()

Returns the number of queued tasks waiting for execution.

A high queued task count indicates that the scheduler is under heavy load and tasks are waiting for execution slots.

Example:

final queuedCount = scheduler.getQueuedTaskCount();
if (queuedCount > 0) {
  print('$queuedCount tasks waiting for execution slots');
}

Implementation
